# Quickstart: File Upload

**Branch**: `002-file-upload`

## Additional Slack Scope Required

Before using `files_upload`, add `files:write` to your Slack App's Bot Token Scopes:

1. Go to [api.slack.com/apps](https://api.slack.com/apps) → your app
2. **OAuth & Permissions → Bot Token Scopes** → Add `files:write`
3. Re-install the app to the workspace (the token itself does not change, but the scope is updated)

## Example invocations

### Upload a PDF without a comment
```json
{
  "tool": "files_upload",
  "arguments": {
    "file_path": "/tmp/reports/quarterly.pdf",
    "channel_id": "C12345678"
  }
}
```

### Upload a PNG with a custom name and comment
```json
{
  "tool": "files_upload",
  "arguments": {
    "file_path": "/tmp/generated/chart_20240317.png",
    "channel_id": "C12345678",
    "filename": "revenue-chart-q1.png",
    "initial_comment": "Here is the Q1 revenue chart generated by the analytics agent."
  }
}
```

### Expected successful response
```json
{
  "id": "F12345678",
  "name": "quarterly.pdf",
  "permalink": "https://myworkspace.slack.com/files/U001/F12345678/quarterly.pdf",
  "channel": "C12345678"
}
```

## Troubleshooting

| Symptom | Cause | Fix |
|---------|-------|-----|
| `File not found: /path/to/file` | Path doesn't exist on server filesystem | Verify the path is correct and the server process can read it |
| `missing_scope` error | Bot missing `files:write` scope | Add scope in Slack App settings and reinstall |
| `channel_not_found` | Wrong channel ID | Use `channels_list` to get a valid ID |
| `not_in_channel` | Bot not a member of the channel | Invite the bot to the channel first |
| `file_too_large` | File exceeds Slack's limit | Slack's limit is 1 GB per file |
